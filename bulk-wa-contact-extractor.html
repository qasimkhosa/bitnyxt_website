Bulk WA Contact Extractor to Excel

A Chrome extension for WhatsApp Web that exports contacts, group members, chat list rows, and labels into Excel workbooks or CSV files from one sidebar workspace.

Single purpose

Export selected WhatsApp Web data into spreadsheet-friendly files

Bulk WA Contact Extractor to Excel is designed for one job: help the user export data they can already access inside WhatsApp Web into structured Excel or CSV output for CRM import, review, and internal workflows.

  • Exports saved or unsaved contact numbers when available
  • Supports group members, chat list rows, and labels
  • Generates spreadsheet-ready output for Excel workflows

Features

What the extension exports

The extension runs only on WhatsApp Web and helps the user export data they explicitly choose to capture from the current workspace.

CT

Contacts

Export saved or unsaved contact details visible in WhatsApp Web into structured rows for spreadsheet review or CRM import.

  • Phone numbers when available
  • Contact and group names
  • Spreadsheet-ready output
GM

Group members

Extract group member lists, including admin state where visible inside the current group context.

  • Group member export
  • Admin state when available
  • Excel and CSV support
LB

Chat list and labels

Capture chat list rows, labels, and recent summaries to support follow-up organization and operations workflows.

  • Labels when available
  • Recent message summaries
  • Sidebar-based workflow
Permissions

Why the extension needs its permissions

tabs

Used to find, focus, open, and message the correct WhatsApp Web tab after the user clicks the extension icon.

storage

Used to persist sidebar state, license state, and service-worker recovery state between sessions.

https://web.whatsapp.com/*

Required so the content script can run on WhatsApp Web and extract the data the user has chosen to export.

https://api.polar.sh/*

Required to validate Polar license keys for Pro access and manage license-related product features.

Privacy summary

Data use is limited to the export workflow

The extension accesses WhatsApp Web content necessary for the user-requested export flow, such as contact or group names, phone numbers when available, labels, recent summaries, member state, business status when available, and notes stored in WhatsApp Web local IndexedDB.

  • Stores local UI state and local trial or Pro state
  • Stores user-entered Polar purchase email and license key for validation
  • Does not describe itself as collecting unrelated browsing history or payment card details
